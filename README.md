CHALLENGE YANG DIBERIKAN :
- bikin filter dan search bar saat ambil course
- bikin grafik di student dashboard dan pet bisa bereaksi sesuai nilai

Projek ini tentang 

Q1 Assignment 1
Fitur 1 = 
- Login, Register dan Logout
- Masuk sesuai role admin dan student

Q2 Assignment 2
Fitur 2 =
- Admin dan Student bisa memakai fiturnya masing-masing
- Admin bisa add & delete course
- Admin bisa add & delete materials
- Admin bisa add & delete quiz
- Student bisa pilih course dan mengerjakan quiznya
- Student bisa menambah dan mencoret selesai to-do list 

Q3 Assignment 3
Fitur 3 =
- Register diperbaharui dengan fitur pilih binatang favorit sebagai pet
- Student punya pet dan bisa menyelamatkan pet di quiz
- Pet bisa digerak-gerakin dalam air
- Tambah css untuk register.php, login.php, student_dashboard.php, quiz.php, dan footer.php
- Pet muncul di dashboard student dan dielus-elus
- to do list yang sudah pernah dibikin bisa di delete
- pet container boxnya bisa digerak-gerakin

Q4 Assignment 4
Fitur 4 =
- Membuat edit materi, course, quiz untuk admin 
- Selesai Challenge 1 yaitu membuat filter dan search bar saat ambil course(tambahan : filter dan search bar juga ada di student dashboard)
- Membuat profile untuk admin dan student untuk melihat statistik keseluruhan dan mengedit username, nama lengkap, dan password
- Setelah mengerjakan quiz bisa di melihat nilainya dan nanti pet bisa react sesuai nilai yang kamu dapatkan, kalau bagus dia bangga, kalau jelek dia sedih
- Selesai Challenge 2 yaitu membuat grafik untuk setiap nilai yang didapatkan dan pet bisa react sesuai nilai yang didapatkan, kalau bagus dia senang, kalau jelek dia menghiburmu, pet akan react ketika hanya melihat grafiknya, pet akan react setiap 7 detik
- Membuat guest dashboard untuk guest, jadi mulai awalnya itu di guest dashboard
